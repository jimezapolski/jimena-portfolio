## Casos de estudio

### 001 · Zenda — AR Education
App multimedia para la interpretación de la Ley de Etiquetado Frontal de Argentina. Desde investigación con 108 usuarios y 2 nutricionistas hasta desarrollo nativo en **React Native + Swift (VisionKit)**. Tesis de grado aprobada con distinción en Universidad Maimónides 2025.

`Ed-Tech` `AR / Realidad Aumentada` `React Native` `Swift` `UX Research` `Tesis`

### 002 · ZonaProp — AI Redesign
Concept redesign de la plataforma inmobiliaria líder de Argentina, incorporando un asistente de IA conversacional y rediseñando el flujo de búsqueda de propiedades desde cero.

`AI / LLM` `Product Redesign` `Real Estate` `UX/UI`

### 003 · Nami — Mental Health
App mobile para acompañar episodios de ansiedad en tiempo real. Diseñada específicamente para el momento de crisis aguda — no para la meditación preventiva. Incluye Storm Mode, personaje animado con 5 estados emocionales y onboarding conversacional.

`Mental Health` `iOS` `Design Thinking` `UX/UI` `Design Challenge`

## Stack del sitio

- HTML + CSS + JavaScript vanilla
- [GSAP](https://gsap.com/) + ScrollTrigger para animaciones
- [Lenis](https://lenis.darkroom.engineering/) para scroll suave
- Sistema de i18n propio ES / EN (sin dependencias)
- Transiciones entre páginas con overlay animado
- Diseño responsive — mobile first
